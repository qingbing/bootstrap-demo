    p
    p.lead : 段落突出
    mark : 内联文本高亮
    .well(默认、.well-lg、.well-sm) : 框选文本
  • 文本对齐和阶段
    .text-left、.text-center、text-right : 文本对齐
    .text-overflow ： 自动截断
  • 大小写转化
    .text-lowercase : 全部小写（对英文）
    .text-uppercase : 全部大写（对英文）
    .text-capitalize : 首字母大写（对英文）
  • 字体效果
    s、del : 删除效果
    u、ins : 下划线效果
    b、strong : 加粗效果
    i、em : 斜体效果
    small : 小号文本效果
  • 特殊
    abbr[title=xx] : 普通提示效果
    abbr.initialism[title=xx] : 小字号提示效果
    address : 地址
